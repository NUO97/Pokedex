This assignment has a lot of support files -- we provide all the HTML and CSS you need for the assignment, and the rest of the support files are images.

The resources.zip file has a copy of everything that you need for the assignment.

Unzip that file, and you will have a copy of the starter directory, with all of the images, and the skeleton HTML and provided CSS file.

Inside your working directory, you should have a directory for icons, images, and sprites, as well as the provided HTML and CSS files.
